# Furora.Media
React Redux Progressive + Universal web app Portfolio site, powered by node + express. Scss based on inuitcss, webpack used to create Service worker.

##Install dependencies
```js
npm install 
```

##Run
```js
npm start
```

##Build
```js
npm run build
```

##Dev
```js
npm run client
```
